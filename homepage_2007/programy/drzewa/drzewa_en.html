AVL Tree Visualization Applet

About

"Drzewa" Applet is designed to graphic presentation of a balanced trees. It allows to observe changes in such tree after inserting an arbitrary element. For now I implemented only AVL tree, but adding new tree kinds shouldn't be difficult now. (TODO: add some new tree kinds finally..)

Manual

  1. Enter number to text field
  2. Press "Wstaw" button to insert element to tree
  3. Press "AVL" button to balance the tree (you can see rotation if it's needed to balance the tree)
  4. Go to point 1

Attention: to get proper AVL tree you must press "AVL" button every time after adding new element!

If you don't care about proper AVL tree, you can ignore "AVL" button. The tree will behave like ordinary BST tree (Binary Search Tree). You can also do rotation by hand by clicking on elements in tree. And you can move whole tree on applet by clicking on some free space inside applet. If some elements overlaps others, just move the tree anywhere and it will fix itself automatically.
