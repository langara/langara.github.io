program "Rekrutacja"

O programie

"Rekrutacja" to program służący do zarządzania bazą danych o pracownikach. Zawiera wszystkie typowe opcje dla takich programów, czyli: wprowadzanie, edycja, usuwanie danych; sortowanie; wyszukiwanie; drukowanie raportów, wyników wyszukiwania, drukowanie różnych formularzy (można definiować nowe); eksport tabel i formularzy do html.

Program został napisany w języku C++ z użyciem biblioteki wxWidgets. Działa pod systemami operacyjnymi MS Windows i Linux (TODO: sprawdzić jak pod Mac OS X).
