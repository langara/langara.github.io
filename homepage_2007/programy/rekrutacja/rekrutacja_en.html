program "Recruitment"

About

"Recruitment" is a program which manages employee data base. It contains all typical functions for such programs like a: insert, edit, delete data; sort; search; printing of: reports, search results and various forms (one can define new ones); export of: table and forms to html file.

Program was written in C++ language with wxWidgets library. It works both on MS Windows systems and on Linux systems (TODO: try to compile it on Mac OS X).
